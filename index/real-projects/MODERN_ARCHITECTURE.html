MODERN ARCHITECTURE

Проект: MODERN ARCHITECTURE

Объект: трехкомнатная квартира

Локация: Беларусь, г.Минск микрорайон Стелла

Площадь: 95 м.кв.

Задача: Легкий в восприятии интерьер с элементами неоклассики

Дизайн-проект интерьера двухкомнатной квартиры. Помещение расположено в многоэтажном многоквартирном крупном жилом комплексе в городе Минске. Квартира разместилась на 25 этаже с видом на город, очень светлая и просторная. Но несмотря на этот фактор, заказчики пожелали видеть свой интерьер в светлых тонах. В дизайн проекте квартиры мы использовали белый цвет, преобладают бежевые, серые и песочного цвета, оттенки.

Попадая в жилое пространство, нас приветствует подвесная тумба с зеркалом на всю высоту стены. Подвесной комод крепится на несущую стену с помощью завесов Camar. Такая консоль выдерживает вес самой себя, имеет два выдвижных ящика, в качестве фурнитуры - система Blum Tandem полного выдвижения с доводчиками. Украшением входной зоны является композиция из верхних подвесных светильников из стекла. Подвесы расположились на разном уровне по высоте. Потолочные светильники излучают белый теплый свет и дают естественное освещение, наполняя интерьер уютом.

Напольное покрытие в проекте – ламинат Tarket 33 класса. Плинтус напольный из мдф, окрашен водоэмульсионной краской белого цвета. Стены – обои на флизелиновой основе. Двери из массива ольхи белого цвета. Стены на лоджии выполнены под покраску, окрашены краской Caparol.

Зона ТВ оформлена в виде подвесной тумбы белого цвета. В качестве декора подобраны настольные лампы с черными абажурами в стиле неоклассики.

Напротив телевизионной зоны расположился мягкий угловой диван с обивкой из ткани ванильного оттенка. Цвет дивана подобран исходя из общего цветового решения интерьера. Дизайн-проект полностью разработан в светлых тонах. За диваном заняла свое место барная стойка со столешницей из акрилового камня черного цвета.

Барная стойка опирается на простенок, который мы возвели во время строительных работ между двух колонн. Одна из колонн задумана дополнительной конструкцией для симметрии. Колонны в составе с барной стойкой служат перегородкой, с помощью которой выполняется зонирование между кухней и гостиной.

Кухня изготовлена по индивидуальному проекту с фасадами из акриловых плит фирмы AGT. Кромка ПВХ в цвет фасадов. Фасады белого цвета на кухне всегда выглядят легко и не громоздко.

Фартук в зоне кухни из керамогранита с ярко выраженной фактурой. Керамогранит белого цвета имеет глянцевую поверхность, отражая предметы в помещении, тем самым, дает больше воздуха в пространстве.

Гарнитур кухни представляет собой Г-образную композицию. Г-образная кухня с пеналами по бокам является полноценным архитектурным ансамблем для квартиры в данном дизайн-проекте интерьера. В одном из шкафов мы разместили встроенный холодильник, во втором - духовой шкаф, под которым поставили не встраиваемую стиральную машину. За одним из фасадов заняла свое место посудомоечная машина шириной 45 см. Ручки мебели строгих прямолинейных форм, в цвете полированный хром. В центре зоны кухни разместилась столовая группа с круглым раскладным столом и стульями белого цвета. Подсветка кухни выполнена с помощью врезного алюминиевого профиля и светодиодной ленты LED.

Интерьер спальни получился с более яркими акцентами. Это нежные оттенки персикового и розового пудрового цвета. Кровать в центре комнаты с изголовьем с втяжками и пуговицами. Интерьер спальни завязан на деталях - прикроватные тумбочки с изящными ручками, рисунок на обоях, подхваты для штор, ткани и декоративные подушки, светильники, спускающиеся с потолка.

По длинной стороне комнаты разместился вместительный шкаф для вещей. Фасады выполнены из ДСП, на который наклеено стекло лакобель. Система раздвижения выбрана бренда Hettich (Германия). Шкаф-купе в спальне - система с доводчиками Hettich top-line.

В качестве основной отделки использовали материалы европейских производителей.

В санузле выдержана аналогичная гамма бежевых и серых тонов. Центр композиции - это подвесная тумба с умывальником и зеркалом с фацетом. Умывальник бренда Roca овальной формы.

Столешница над тумбой из постформинга Egger белого цвета. Толщина постформинга 38мм. Фасады для тумбы изготовлены из МДФ крашеного по RAL с глянцевой поверхностью. Выдвижные системы Blum Tandem полного выдвижения с доводчиками.
